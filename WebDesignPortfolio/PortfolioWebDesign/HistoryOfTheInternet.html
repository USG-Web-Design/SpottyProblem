Web 1

  • Only readable, not interactable
  • At this stage, websites are basically static.
  • There were many major sites, services, informational pages, and other one-way communication webpages.

Web 2

  • At the web 2.0 stage, the internet became dynamic and listened to user input. (Ex: AI)
  • Decentralization, blockchains, metaverses, digital infrastructure.
  • Current Web under development.






Web 3

  • Web Tech evolves into a usage that allows people to create, share, and connect content via search and analysis through words instead of numbers.
  • Websites at this stage are basically static, as there is no dynamic rendering.
  • There were major new sites, products, services information pages, and other one-way communication websites.
